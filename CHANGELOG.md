# Changelog

All notable changes to this dotfiles repository will be documented in this file.

## [2026-04-15]

### Added

- Added local `agents/` configuration for Codex, including reusable skills for commit workflows, brainstorming, debugging, API design, interface design, changelog generation, and React best practices.
- Added a dedicated `tmux` setup with `C-s` as prefix, Vim-style pane navigation, popup session switching through `fzf`, and a Catppuccin-based status line.
- Expanded `zsh` with faster day-to-day aliases for Git, Docker Compose, Kubernetes, Terraform, and directory navigation.

### Changed

- Simplified the `Ghostty` terminal configuration to focus on split navigation, clipboard shortcuts, copy-on-select behavior, and a cleaner borderless window setup.
- Updated the `Starship` prompt icon and spacing for a cleaner prompt header.
